# Pollution Control

- Agent internet should be disabled in the benchmark run.
- The task does not ask for a known public benchmark answer. Hidden eval cells and labels are generated by a private deterministic augmentation process.
- Public data sources are used only as calibration references and raw provenance. The hidden labels are not copied from public datasets.
- The split is by cell ID. Visible train/dev cells cannot directly reconstruct hidden eval labels.
- The scorer and hidden labels live outside `agent-start/` and must be included only in the judge package.
- `submit_paths` should include only `train.py`, `predict.py`, `src/`, and `requirements.txt`.
